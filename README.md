# 梁熙涵的拼豆图纸生成器

一个基于网页的拼豆图纸生成工具。用户上传像素图片后，系统自动将颜色匹配到 Perler 官方65色板，生成带网格线和色号标记的拼豆图纸，支持 PDF/PNG 下载，并支持手动修改任意格子的颜色。

## 功能特性

- 上传图片自动生成拼豆图纸
- Perler 官方65色板 + CIEDE2000 高精度色差匹配算法
- 带网格线和色号标记的图纸输出
- 支持下载为 PNG 图片或 PDF 文件
- 支持点击图纸手动修改任意位置颜色
